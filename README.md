# Back-end Finalizando do App

Nessa aula terminaremos algumas das funcionalidades faltantes, entre elas a listagem do prestador para verificar todos os agendamentos que ele tem por dia. Lembrando dos nossos requisitos:

## Installation

na pasta raiz do projeto usar os comandos ```npm install```ou  ```yarn```
  para instalar todos os pacotes do package.json

## Estrutura

A pasta Shared contem todos os modulos que podem ser compartilhados dentro da aplicação.
A pasta modules, é onde fica os modulos que individuais de cada funcionalidade.
Basicamente é o modulos de agendamentos, usuarios e notificações.
todos os modulos compartilham de uma mesma estrutura de divisão de arquivos.
